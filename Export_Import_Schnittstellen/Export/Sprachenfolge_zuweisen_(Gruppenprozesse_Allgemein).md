# Sprachenfolge zuweisen (Gruppenprozesse Allgemein)

Dieser Gruppenprozess ermöglicht es, allen Schülern in der aktuellen
Auswahl eine Sprache zuzuweisen.Unter **Sprache** werden alle Fremdsprachen, die man unter
Schulverwaltung als Unterrichtsfach deklariert hat, zur Auswahl
angeboten.Unter **Modus** muss man angeben, ob es sich um einen *Neueintrag*, eine
*Aktualisierung* oder eine *Löschung* handelt.Weiterhin muss man einen Eintrag bei **Reihenfolge** und dem **Beginn-**
bzw. **End**-Jahrgang und Abschnitt vornehmen. Ein Eintrag beim
*Referenzniveau* ist nicht nötig, dieses kann beim Abschluss der Sprache
aus den Leistungsdaten, auch per Gruppenprozess für große Schülermengen,
erzeugt werden.
