# Für Schulträger (Verwaltung Export)

![SchILD_Export_Schulträger_1.png](../graphics/SchILD_Export_Schulträger_1.png)

 Über *Verwaltung ➜ Export* ➜ **Für Schulträger** können Sie
einen Export für Daten an den Schulträger anstoßen.  

![SchILD_Export_Schulträger_2_Optionen.png](../graphics/SchILD_Export_Schulträger_2_Optionen.png)

 Wählen Sie zuerst ein **Ausgabeverzeichnis**.Anschließend können unterschiedliche *Optionen* festgelegt werden.Wählen Sie die Art der Verschlüsselung.-   **Daten verschlüsselt mit Standardkenntwort** sorgt dafür, dass die
    Daten mit dem Schulträgermodul entschlüsselt werden können. In
    diesem Fall muss kein Kenntwort gesetzt und ausgetauscht werden.
-   **Daten verschlüsselt mit individuellem Kennwort** lässt Sie ein
    eigenes Kennwort setzen. Dieses Kennwort muss dann auf sicherem Wege
    dem Schulträger mitgeteilt werden und darf keinesfalls zusammen mit
    einer eventuellen Email mit verschickt werden, über die auch die
    Daten übermittelt werden. Das Kennwort soll modernen Standards in
    Länge und der Verwendung von Groß-, Kleinschreibung, Zahlen und
    Sonderzeichen entsprechen.Damit Sie sicher sind, das Kennwort auch so wie gedacht einzutippen,
lässt sich bei der Eingabe das **Kennwort anzeigen**.Wurde der Emailversand in SchILD-NRW eingerichtet, lässt sich eine
**Emailadresse** angeben, an welche der Export direkt versandt wird.Schlussendlich wählen Sie, welche Daten exportiert werden sollen:-   **Alle Schüler** exportiert die komplette Schule.
-   **Nur Schüler, die in der aktuellen Auswahl enthalten sind**
    exportiert die links angezeigten Schüler. Diese werden vor Aufrufen
    des Exportfensters über den Reiter *Schüler* gefiltert.**Export starten** speichert eine Datei *\[Schulnummer\].zip* in das
angegebene Verzeichnis.

Diese kann mit dem verwendeten Passwort beziehungsweise über das
Schulträgermodul entpackt werden.

Das Format entspricht den Schnittstellendateien und kann in einem
Tabellenkalkulationsprogramm weiterverarbeitet werden.
