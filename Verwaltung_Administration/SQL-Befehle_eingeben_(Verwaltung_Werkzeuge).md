# SQL-Befehle eingeben (Verwaltung Werkzeuge)

![SchILD_Werkzeuge_SQL_1.png](../graphics/SchILD_Werkzeuge_SQL_1.png)

 

![SchILD_Werkzeuge_SQL_2_Ergebnis.png](../graphics/SchILD_Werkzeuge_SQL_2_Ergebnis.png)

Sie haben als *Administrator* die Möglichkeit, beliebige SQL-Befehle in
der Datenbank auszuführen.

Diese Funktion ist über *Verwaltung ➜ Werkzeuge ➜ SQL-Befehle eingeben*
zu erreichen.Überlegen Sie bitte sehr genau, ob eine *Veränderung*, zum Beispiel das
Anlegen einer neuen Tabelle oder ein direktes Umsetzen bestimmter Werte,
auch tatsächlich eine Gute Idee™ ist oder ob Sie das noch einmal
überdenken sollten.In außergewöhnlichen Situationen kann es eventuell notwendig sein,
bestimmte Daten direkt zu ändern. Dies ist über dieses Werkzeug möglich.Halten Sie bitte Rücksprache mit Ihrer Fachberatung!
