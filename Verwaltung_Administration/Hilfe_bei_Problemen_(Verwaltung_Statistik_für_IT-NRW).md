# Hilfe bei Problemen (Verwaltung Statistik für IT-NRW)Über *Verwaltung ➜ Statistik für IT.NRW* erreichen Sie Funktionen, die
bei der Vorbereitung und der Erstellung der amtlichen Statistik helfen
können.Der Punkt **Hilfe bei Problemen** dient dazu, das Betriebssystem MS
Windows für die Statistik mit SchILD-NRW einzurichten.Üblicherweise geschieht das bei der Installation von SchILD-NRW, jedoch
kann es je nachdem, wie und auf welchem Rechner SchILD-Installiert
wurde, dazu kommen, dass bestimmte Programmteile nicht korrekt
registriert wurden.

![Hilfe_bei_Problemen.jpg](../graphics/Hilfe_bei_Problemen.jpg)

 Wird dieser Menüpunkt aufgerufen, bietet das Fenster eine
umfangreiche Erklärung.Sollte beim Aufrufen der Gesamtprüfung von SChILD-NRW oder der
Erstellung der Dateien für den Export zu ASDPC eine Fehlermeldung
erscheinen, dann liegt das häufig an der fehlenden Registrierung
bestimmter Dateien bei einer Neuinstallation von SchILD.

Diese Registrierung muss in einem Netzwerk auf jedem Rechner, der einen
SchILD-Klienten *ausführt*, einzeln erfolgen. Es reicht nicht, wenn dies
nur auf einem Rechner geschieht. Daher kann es dazu kommen, dass einige
Computer der gleichen Serverumgebung problemlos eine Gesamtprüfung
durchführen können und andere nicht.

Dies bedeutet auch: Rechner, mit denen die Statstik-Prüfung nicht
ausgeführt wird, benötigen keine gesonderte Registrierung der Dateien.Wie in der Fehlermeldung beschrieben, muss sich nun der
System-Adminstrator des jeweiligen Computers, also *nicht* der
SchILD-Admin, sondern der Admin für Windows, als lokaler Admin
einloggen.Ohne lokale Admin-Rechte auf dem Windows-Rechner bleibt die Schaltfläche
ausgegraut und kann nicht angeklickt werden. 

![Dateien_registrieren.jpg](../graphics/Dateien_registrieren.jpg)

Wird **Dateien registrieren** angeklickt, führt SchILD-NRW diese
Operation aus.Es öffnen sich mehrere Fenster, die jeweils mit **Ok** geschlossen
werden können.  

### SchILD-NRW als Windows-Administrator starten

![SchILD_Verwaltung_Statistik_DLLsRegistrieren_AdminWin.png](../graphics/SchILD_Verwaltung_Statistik_DLLsRegistrieren_AdminWin.png)

 Das Starten von SchILD-NRW als Windows-Administrator
erfolgt durch das Anklicken des SchILD-Icons der *SchILD30.exe* mit der
*rechten Maustaste* und dem Anklicken von **Als Administrator
ausführen**.Dann geht es wie oben ausgeführt mit *Verwaltung ➜ Statistik für IT.NRW
➜ Hilfe bei Problemen* weiter. Klicken Sie nun auf **Dateien
registrieren** klicken.
