# Abiturjahrgänge einrichten

## Einrichtung der Jahrgänge

Durch "+" können die aktiven Jahrgangsstufen der Oberstufe sowie eine
ggf. neu zu planende EF hinzugefügt werden. 

![SVWS_Oberstufe_Jahrganganlegen_1.png](../../graphics/SVWS_Oberstufe_Jahrganganlegen_1.png)

 

![SVWS_Oberstufe_Jahrganganlegen_2.png](../../graphics/SVWS_Oberstufe_Jahrganganlegen_2.png)

Es können nun die **allgemeine Vorlagen** oder individuell **die
jeweilige Jahrgangsstufe** bearbeitet werden hinsichtlich:1.  Fächereigenschaften
2.  Ausschlussregeln
3.  Weitere Kombinationsbedingungen
4.  Beratungslehrkräfte
5.  ...
