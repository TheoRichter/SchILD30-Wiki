# Startklar (Verwaltung Export)

![SchILD_Export_Starklar.png](../graphics/SchILD_Export_Starklar.png)

Über *Verwaltung ➜ Export* ➜ **Startklar** lässt sich eine Datei für
*Startklar!* exportieren.Wählen Sie ein Verzeichnis, in das exportiert werden soll. Hier im
Screenshot wurde ein Verzeichnis *Export* im *SVWS-Arbeitsverzeichnis*
gewählt.Klicken Sie auf `Daten exportieren`.  
Es wird eine Textdatei *Startklar_SCHULNUMMER.txt* erstellt, in der sich
eine Kopfzeile befindet, welche die jeweilige Datenart kenntlich macht.
Dann folgen die tatsächlichen Daten, die jeweils von einem ";" getrennt
werden:` Nachname;Vorname;Geburtsdatum;Schuljahr;Klasse;Parallelität;Geschlecht;Anmerkungen;Staatsangehörigkeit;Förderbedarf`  
` "Abel";"Sina";"15.05.2004";"2020/2021";"10";"c";"w";"";"1";"nein"`  
` "Adam";"Marie";"23.04.2005";"2020/2021";"10";"d";"w";"";"1";"nein"`  
` "Adams";"Aaliya";"16.05.2005";"2020/2021";"10";"a";"w";"";"1";"ja"`
