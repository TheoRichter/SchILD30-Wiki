# Zeichentabelle von Windows aufrufen (Verwaltung Werkzeuge)

![SchILD_Verwaltung_Werkzeuge_Zeichentabelle.png](../graphics/SchILD_Verwaltung_Werkzeuge_Zeichentabelle.png)

 Sollte ein Symbold benötigt werden, das nicht direkt über
die Tastatur abrufbar ist, lässt sich von SchILD-NRW aus die
Windows-Zeichentabelle aufrufen.

Diese findet sich über *Verwaltung ➜ Werkzeuge* ➜ **Zeichentabelle von
Windows aufrufen**.Um noch weitere Auswahlmöglichkeiten zu haben, lässt sich die
*Erweiterte Ansicht* aktivieren.Suchen Sie nun Ihr Zeichen und fügen sie es ein beziehungsweise kopieren
Sie es in den Zwischenspeicher.
