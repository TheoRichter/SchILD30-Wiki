# Kennwort ändern (Verwaltung Schule)

![SchILD_Verwaltung_KennwortÄndern.png](../graphics/SchILD_Verwaltung_KennwortÄndern.png)

Mit dem Knopf *Verwaltung ➜ Benutzer* ➜ **Kennwort ändern** kann ein
eingelogter Benutzer das eigene Kennwort ändern.Geben Sie zuerst das alte, dann zweimal das neue Kennwort an.Bestätigen Sie mit **OK**.
