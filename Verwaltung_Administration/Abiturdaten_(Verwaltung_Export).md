# Abiturdaten (Verwaltung Export)

![Schild_Verwaltung_Export_Abiturdaten.png](../graphics\Schild_Verwaltung_Export_Abiturdaten.png)

 Über *Verwaltung ➜ Export* ➜ **Abiturdaten** lässt sich
eine Textdatei mit den Daten zum aktuellen Abitur erzeugen.Hierzu ist ein **Ausgabeverzeichnis** zu wählen.Klicken Sie dnn auf `Daten exportieren`. Hierbei werden existierende
Daten überschrieben, die diesbezügliche Meldung ist mit `OK` zu
bestätigen.  
In diesem Ordner wird eine Datei *AbiturExport.txt* erzeugt, in der die
Kopfzeile wie folgt aussieht und damit in den Folgezeilen auch die
entsprechenden Informationen enthält.` SchulNR;SchuelerNr;Geschlecht;Fach;Kursart;Fremdspr_fortgef;Vornote;PruefNote;AbiNote;Bestanden;PruefOrdn;SGL`Es wird für jedes Abiturfach eine eigene Zeile erzeugt.
