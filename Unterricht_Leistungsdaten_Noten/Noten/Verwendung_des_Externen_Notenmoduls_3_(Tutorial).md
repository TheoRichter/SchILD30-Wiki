# Verwendung des Externen Notenmoduls 3 (Tutorial)

Das **Externe Notenmodul 3** dient dazu, aus SchILD-NRW 3 exportierte
Notendateien für die Lehrkräfte zur Verfügung stellen zu können, so dass
jede Lehrkraft ihre Noten unabhängig von einem SchILD-NRW-Zugang
einzutragen. Das Externe Notenmodul kann ebenfalls für Floskeln,
Teilleistungen, Fehlstunden und so weiter verwendet werden.



Derzeit ist das Externe Notenmodul 3 als Betaversion
markiert.



## Übersicht (für die Artikelplanung)
1.  Zuerst müssen die in **Schüler ➜ Akt. Halbjahr ➜ Leistungsdaten**
    (ohne die Noten) der Schüler korrekt befüllt werden, daher es müssen
    alle Fächer mit den *aktuell korrekten Lehrkräften* eintragen sein.
2.  Stellen Sie sicher, dass alle Lehrkräfte Initialkennworte gesetzt
    haben (siehe unten).
3.  Die Notendateien werden aus SchILD-NRW exportiert und...
4.  ... an die Lehrkräfte verteilt.
5.  Die Lehrkräfte laden das ENM herunter/installieren es, nutzen das
    ENM3 um ihre Noten einzutragen und übertragen die Notendateien
    gegebenenfalls an die Schule zurück.
6.  Die Notendateien werden zurück in SchILD-NRW importiert.
