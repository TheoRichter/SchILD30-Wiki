# Aufnahmebestätigung (Verwaltung Import)

Über SchülerOnline besteht die Möglichkeit für abgebende Schulen, eine
Datei mit Schulwechslern zu exportieren.Hierbei wird eine Datei *Aufnahmen_SCHULNUMMER_DATUM.txt* erzeugt.

Diese Datei kann in SchILD-NRW über *Verwaltung ➜ Import* ➜
**Aufnahmebestätigungen** eingelesen werden.
